# 🔷 3D Сечения Геометрических Фигур

Интерактивное веб-приложение для визуализации и построения сечений трёхмерных геометрических фигур.

## 📖 Описание

Это одностраничное приложение позволяет изучать и исследовать сечения различных геометрических тел (куб, пирамида, призма, тетраэдр) плоскостью. Приложение предоставляет интерактивную 3D-визуализацию с возможностью вращения, масштабирования и редактирования секущей плоскости.

## ✨ Возможности

### Поддерживаемые фигуры
- **Куб** — правильный гексаэдр
- **Пирамида** — четырёхугольная пирамида
- **Призма** — прямая призма
- **Тетраэдр** — правильный тетраэдр

### Методы построения сечений
- **Метод следов** — построение линии пересечения секущей плоскости с гранями
- **Метод вспомогательных плоскостей** — использование дополнительных плоскостей для нахождения точек сечения
- **Пошаговое построение** — анимированное отображение процесса построения сечения

### Интерактивные функции
- 🖱️ **Вращение** — перетаскивание мышью для вращения фигуры
- 🔍 **Масштабирование** — колесо мыши для изменения масштаба
- ✏️ **Редактирование точек** — перетаскивание точек для изменения положения секущей плоскости
- 🎨 **Визуальные эффекты** — подсветка активных элементов, прозрачность сечения
- 📝 **Подписи вершин** — отображение буквенных обозначений вершин и точек сечения

## 🚀 Использование

Откройте файл `section.html` в любом современном браузере:

```bash
# Просто откройте файл в браузере
open section.html  # macOS
xdg-open section.html  # Linux
start section.html  # Windows
```

## 🎮 Управление

| Действие | Управление |
|----------|------------|
| Вращение фигуры | Левая кнопка мыши + перетаскивание |
| Масштабирование | Колесо мыши |
| Перемещение точки | Перетаскивание точки мышью |
| Выбор фигуры | Кнопки в верхней панели |
| Настройка видимости | Чекбоксы в панели управления |
